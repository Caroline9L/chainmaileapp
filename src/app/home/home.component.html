What It Does

It's happened to all of us. Put in a big ring order...and forget why you ordered those 18g 5/32" rings. (Not me, I make everything with those.) Or ordered some so long ago that their purpose has been lost to the ages. Maybe you just have a lot of leftovers and a desire to experiment, but aren't sure what you can do with the sizes you have in stock. Or, you want to try a particular weave, but don't know if your existing rings will work.

Enter The Weave Finder! Simply enter the ring sizes you are interested in using (awg only at this time), and TWF will return the weaves that can utilize that size.

This project is a work in progress. Currently TWF will only search a single size of ring, and weaves are limited. As my skills with this framework expand, I hope to extend the boundaries to include more weaves and a more robust search function. For now, this is just for funsies.
